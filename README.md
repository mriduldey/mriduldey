A provider-neutral TypeScript SDK for working with source-code repositories through stable contracts. It provides one API for repository files, branches, commits, issues, pull requests, releases, and tags, with GitHub as the first provider.

`TypeScript` · `SDK Design` · `Provider Architecture` · `Testing` · `Developer Experience`
